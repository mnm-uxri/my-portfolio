## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- Angular CLI
```bash
npm install -g @angular/cli
```

### Installation
```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
npm install
```

### Development Server
```bash
ng serve
```

Visit `http://localhost:4200/` — the app auto-reloads on file changes.

---

## 🧱 Building
```bash
ng build
```

Build artifacts are output to the `dist/` directory. Production builds are optimized for performance and speed.

---

## 🧪 Testing

### Unit Tests
```bash
ng test
```

Runs unit tests via [Vitest](https://vitest.dev/).

### End-to-End Tests
```bash
ng e2e
```

> Angular CLI does not include an e2e framework by default — install one that suits your needs.
